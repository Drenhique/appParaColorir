# App Para Colorir #

Este projeto consiste em um protótipo de aplicativo infantil para desenho e pintura digital, desenvolvido como parte da disciplina de Engenharia de Software.

## Funcionalidades

- Área de desenho interativa
- Seleção de cores
- Ferramentas de desenho
- Galeria de desenhos
- Interface simples voltada ao público infantil

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Objetivo

O objetivo do projeto é demonstrar um Produto Mínimo Viável (MVP) de uma aplicação que permita que crianças criem desenhos digitais de forma simples e interativa.
